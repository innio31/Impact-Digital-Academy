Assignment Instructions

Objective: Create a well-labeled, organized Excel workbook to track simple daily expenses. This assignment practices the core skills from Week 1: creating, navigating, formatting, and saving workbooks.

Step-by-Step Instructions:

  1. Create & Save a New Workbook:
    • Open Microsoft Excel and create a new, blank workbook
    • Save this file immediately to your preferred location
    • Use this naming convention:
      LastName_FirstName_Wk1_Expenses.xlsx
  2. Rename the Worksheet:
    • Double-click on the default sheet name "Sheet1"
    • Rename it to: "January_Expenses"
  3. Set Up Your Data & Navigate:
    • In Cell A1, type: "Personal Expense Tracker - January"
    • In Cell A3, type: "Date"
    • Navigate and type headers in these cells:
      B3: "Description"
      C3: "Category"
      D3: "Amount ($)"
  4. Enter Sample Data:
    • Starting in Row 4, enter at least 5 fictional expense transactions
    • Use different categories: Food, Transportation, Entertainment, etc.
    • Make sure to format dates properly (e.g., 1/15/2026)
  5. Apply Basic Formatting:
    • Select the range A3:D3 (your headers)
    • Go to the Home ribbon tab
    • Make headers Bold (click "B" icon or press Ctrl+B)
    • Apply a Fill Color to headers (choose any light color)
  6. Add a Simple Hyperlink:
    • In Cell A10, type the text: Go to Top
    • Select Cell A10
    • Go to Insert â†’ Link or press Ctrl + K
    • Choose "Place in This Document"
    • Type cell reference: A1 and click OK
  7. Final Save and Submit:
    • Save your work one final time (Ctrl + S)
    • Upload the completed file using the submission form below
