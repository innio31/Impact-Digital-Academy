Part 1: Format a Monthly Sales Report

Hands-On Exercise: Professional Sales Report

Objective: Apply Week 2 skills to create a professional, printable sales report.

1 Open Excel and create a new workbook

Start Microsoft Excel 2019 and create a new blank workbook.

Keyboard Shortcut: Ctrl + N creates a new workbook
2 Enter the sales data in the following structure:
Month Sales Expenses Profit
January 15000 9000 6000
February 18000 9500 8500
March 22000 11000 11000
Total =SUM(B2:B4) =SUM(C2:C4) =SUM(D2:D4)
Average =AVERAGE(B2:B4) =AVERAGE(C2:C4) =AVERAGE(D2:D4)
Tip: Enter the data exactly as shown. The formulas in the Total and Average rows will calculate automatically.
3 Apply professional formatting
  1. Bold the header row: Select row 1 (A1:D1) and press Ctrl + B
  2. AutoFit all columns: Double-click between column headers OR select all columns (Ctrl + A) then double-click any column border
  3. Center align headers: Select A1:D1, then Home tab → Alignment group → Center
  4. Apply Currency format: Select B2:D6, then Home tab → Number group → click "$" Currency symbol
  5. Add borders: Select A1:D6, then Home tab → Font group → Borders → All Borders
AutoFit Shortcut: Select columns, then press Alt + H, O, I
4 Configure Page Setup for printing
  1. Set orientation to Landscape: Page Layout tab → Page Setup group → Orientation → Landscape
  2. Set margins to Narrow: Page Layout tab → Page Setup group → Margins → Narrow
  3. Add a Custom Header:
    • Page Layout tab → Page Setup group → click Dialog Box Launcher (small arrow)
    • Go to Header/Footer tab → Custom Header
    • In the Center section, type: "Sales Report 2024 - Q1 Results"
  4. Add a Footer: In same dialog, Custom Footer → Right section → Insert Page Number (click # icon)
5 View & Print Setup
  1. Freeze the top row: View tab → Window group → Freeze Panes → Freeze Top Row
  2. Set print area: Select A1:D6 → Page Layout tab → Page Setup group → Print Area → Set Print Area
  3. Preview in Page Layout View: View tab → Workbook Views group → Page Layout
  4. Show formulas: Press Ctrl + ` (backtick) to toggle formula display
  5. Use Print Preview: File tab → Print (or Ctrl + P) to see how it will print
Print Preview Tip: In Print Preview, check that all data fits on one page and headers/footers are correct.
6 Save your work

Save your completed sales report with the filename:

Sales_Report_Formatted.xlsx
Save Shortcut: Ctrl + S saves your workbook. Use Save As (F12) to save with a new name.

Part 2: Create a Print-Ready Invoice Template

Professional Business Document
1 Set up invoice header and company information
  1. Create a new worksheet in your workbook (click the + icon next to sheet tabs)
  2. In cell A1, type your company name: "Excel Solutions Inc."
  3. In A2:A4, add company address (use your own or fictional address)
  4. Format company name as bold, size 16, color: Green
  5. Add "INVOICE" in cell F1, formatted as bold, size 18, centered
2 Create customer information section

Set up the following structure starting at row 7:

Bill To: Invoice Details:
Customer Name: [Enter Name] Invoice #: INV-2024-001
Address: [Enter Address] Date: =TODAY()
City, State, ZIP: [Enter City] Due Date: =TODAY()+30
3 Create the invoice items table

Starting at row 15, create this table structure:

Item # Description Quantity Unit Price Total
1 Excel Training Session 2 250.00 =C16*D16
2 Report Template Design 1 450.00 =C17*D17
3 Data Analysis Service 3 180.00 =C18*D18
Subtotal: =SUM(E16:E18)
Tax (8%): =E19*0.08
Grand Total: =E19+E20
Formula Tip: The formulas in the Total column and summary rows will calculate automatically when you enter quantities and prices.
4 Apply professional formatting
  1. Add borders: Apply All Borders to the entire invoice table
  2. Format numbers: Apply Currency format to Unit Price and all total columns
  3. Align text: Center align Item #, Quantity, Unit Price, Total columns
  4. Shade headers: Apply light green fill to all header rows
  5. Bold totals: Make Subtotal, Tax, and Grand Total rows bold
5 Configure for printing
  1. Set print area: Select all invoice content (A1:G25) and set as print area
  2. Add header: Custom header with company name and "INVOICE"
  3. Add footer: Page number, date, and "Confidential" text
  4. Adjust margins: Use Normal margins, center horizontally on page
  5. Preview: Check in Page Layout view and Print Preview
6 Save the invoice template

Save your completed invoice with the filename:

Invoice_Template.xlsx

Part 3: View Management Practice

Mastering Excel Views
1 Open any dataset with many rows/columns

Open your Sales Report or create a new dataset with at least 20 rows and 5 columns of data.

Dataset Suggestion: Create a monthly expense tracker with categories and amounts for 12 months.
2 Practice view management features
  1. Freeze the first column: View tab → Freeze Panes → Freeze First Column
  2. Split the window horizontally: View tab → Split (click Split button)
  3. Remove split: Click Split button again to remove
  4. Switch between views: Try Normal, Page Layout, and Page Break Preview views
  5. Zoom controls: Practice using Zoom In (Ctrl + +) and Zoom Out (Ctrl + -)
View Shortcuts:
• Normal View: Alt + W, L
• Page Layout View: Alt + W, P
• Page Break Preview: Alt + W, I
3 Save custom views and take screenshot
  1. Create Custom Views:
    • Set up your worksheet with specific zoom, frozen panes, etc.
    • View tab → Workbook Views → Custom Views → Add
    • Name it "Data Entry View" and save
    • Change settings and save another as "Print Preview View"
  2. Take a screenshot showing frozen panes and split window
  3. Save screenshot as View_Management.jpg
