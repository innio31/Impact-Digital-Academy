üìå 1. Introduction to Text Editing & Formatting

Text editing modifies content (add, delete, move, correct). Formatting changes appearance to improve readability, hierarchy, and consistency.

  • Editing: altering words, sentences, structure.
  • Formatting: font, alignment, spacing, lists, etc.

Home ‚Üí Paragraph ‚Üí Borders dropdown. Add lines or background color.

‚úÖ 6. Practical Applications & Best Practices

  • Consistency: use Styles, not manual formatting.
  • Readability: line length 50‚Äë75 characters, 10‚Äë12pt body, 1.15‚Äë1.5 spacing.
  • Typography: max 2‚Äë3 fonts, hierarchy via size/weight, avoid ALL CAPS.
  • Lists: parallel structure, consistent punctuation.

Troubleshooting

Issue Likely cause / fix
Extra space between paragraphs Use paragraph spacing instead of double Enter.
Paste brings unwanted format Use "Keep Text Only".
Bullets misaligned Adjust indent in ruler or paragraph dialog.

Exercise 1: Selection & cut/copy

Type a paragraph. Select words, lines, sentences. Cut last sentence, paste to beginning. Copy first sentence to end. Undo.

Exercise 2: Character formatting

Exercise 4: Lists
