üìå 1. INTRODUCTION TO TEXT IN CORELDRAW

CorelDRAW is a powerful vector graphics editor that offers sophisticated text handling capabilities essential for desktop publishing, logo design, brochure creation, and various other design projects. Understanding the text tools thoroughly is crucial for any designer, as text often forms the backbone of communication in design work.

The Importance of Text in Design

  • Text conveys information and messages
  • Typography establishes visual hierarchy
  • Font choices influence brand perception
  • Proper text formatting ensures readability
  • Text integration with graphics creates cohesive designs

CorelDRAW provides two distinct types of text objects, each designed for specific purposes and offering different capabilities.

2.1 Comparison Table: Artistic Text vs. Paragraph Text

‚ú® 3. ARTISTIC TEXT: COMPREHENSIVE GUIDE

Artistic text is designed for creating visually impactful typography for headlines, logos, short phrases, and design elements where text needs to follow creative paths or shapes.

3.1 Creating Artistic Text

Method 1: Text Tool Direct Entry
1. Select the Text Tool (F8) from the Toolbox
2. Click once on the drawing page (do not click and drag)
3. A blinking cursor appears - start typing immediately
4. Press Enter to create a new line

Method 2: Importing Text as Artistic
1. Copy text from external source (Ctrl+C)
2. Select Text Tool and click on page
3. Paste text (Ctrl+V)
4. Text will paste as Artistic Text if content is short

Method 3: Converting Objects to Artistic Text
- Select a curve or shape
- Go to Object > Convert to Artistic Text (only available for compatible objects)

3.2 Formatting Artistic Text

Accessing Formatting Options:
- Property Bar: Most common options available when Text Tool is active
- Text Toolbar: Right-click on any toolbar area and select "Text"
- Format Text Docker: Window > Dockers > Text > Text Properties (Ctrl+T)

Font Selection:
- Click the Font List dropdown on Property Bar
- Scroll or type font name to search
- Fonts are displayed with their actual appearance (if enabled)
- Recently used fonts appear at the top

Font Size:
- Select from predefined sizes in dropdown
- Type custom size directly (supports decimal values)
- Keyboard: Ctrl+Shift+> (increase), Ctrl+Shift+< (decrease)
- Maximum size: 999.999 points (practically unlimited for scaling)

Font Style:
- Regular, Italic, Bold, Bold Italic (availability depends on font)
- Synthetic styles: Available even if font doesn't include specific style

Text Color:
- Select text with Pick Tool or highlight with Text Tool
- Left-click color palette for fill color
- Right-click color palette for outline color
- Drag color from palette onto text for quick application

Character Effects:
- Underline (single, double, thick)
- Strikethrough
- Overline
- Superscript/Subscript
- Small caps/All caps
- Outline, Shadow, Mirror (via Effects menu)

3.3 Advanced Artistic Text Features

Fitting Text to a Path
Method:
1. Create artistic text and a path (line, curve, circle)
2. Select both objects (Shift+click)
3. Go to Text > Fit Text to Path
4. Text attaches to the path with default settings

Adjusting Text on Path:
- Use the Pick Tool to select the text-on-path object
- Property Bar displays specialized controls:
- Text Orientation: Rotate, Skew, or keep text upright
- Distance from Path: Positive/negative values
- Horizontal Offset: Slide text along path
- Mirror Text: Flip horizontally or vertically

Editing Text on Path:
- Double-click text to edit content
- Path remains editable - text updates automatically
- To separate text from path: Arrange > Break Text Apart

Applying Effects to Artistic Text
Artistic text can accept virtually all CorelDRAW effects:
Interactive Effects:
- Drop Shadow: Toolbox > Drop Shadow Tool
- Contour: Toolbox > Contour Tool
- Blend: Toolbox > Blend Tool
- Envelope: Toolbox > Envelope Tool
- Extrude: Toolbox > Extrude Tool
- Bevel: Toolbox > Bevel Tool

Important: Before applying complex effects, consider:
- Effects increase file size
- Some effects cannot be edited as text afterward
- Convert to curves if extensive manipulation needed

Converting Artistic Text to Curves
Process:
1. Select artistic text with Pick Tool
2. Right-click > Convert to Curves (Ctrl+Q)
3. Text becomes editable vector shapes
After Conversion:
- Individual characters become curve objects
- Can edit individual nodes
- Text content no longer editable as text
- Cannot revert to editable text
When to Convert to Curves:
- Creating custom logos
- Extreme distortion requirements
- Sending files to printers (prevents font substitution)
- Creating unique typographic illustrations

üìÉ 4. PARAGRAPH TEXT: COMPREHENSIVE GUIDE

Paragraph text is designed for handling large amounts of text in documents, brochures, newsletters, and any project requiring organized text blocks with professional typographic controls.

4.1 Creating Paragraph Text Frames

Method 1: Drawing a Frame
1. Select Text Tool (F8)
2. Click and drag diagonally to create a rectangular frame
3. Release mouse button - cursor appears inside frame
4. Begin typing or paste text
5. Text automatically wraps to fit frame width

Method 2: Creating from Existing Artistic Text
1. Select artistic text with Pick Tool
2. Go to Text > Convert to Paragraph Text (Ctrl+F8)
3. Artistic text converts with its content preserved

Method 3: Importing Text into Frame
1. Create empty paragraph text frame
2. Right-click frame > Import Text
3. Navigate to text file (.txt, .doc, .rtf, etc.)
4. Text flows into frame

Method 4: Pasting External Text
1. Copy text from external source
2. Create paragraph frame
3. Paste (Ctrl+V)
4. Choose paste options if dialog appears

4.2 Paragraph Frame Types

Fixed Frames:
- Created by click-and-drag
- Size remains constant
- Text overflows if content exceeds capacity

Expandable Frames:
- Created by clicking (not dragging) with Text Tool
- Frame expands vertically as you type
- Width determined by margins set in Options

4.3 Managing Text Flow and Overflow

Understanding Flow Indicators:
- Empty Frame: No text
- Text in Frame: Normal display
- Overflow Indicator: Red arrow in bottom-center handle ‚Äì indicates text exceeds frame capacity ‚Äì must link to another frame or resize frame

Linking Paragraph Frames:
1. Create Link: Select frame with overflow indicator, click the Text Flow tab at bottom of frame, cursor changes to "link" icon, click in empty frame or create new frame by dragging.
2. Navigate Linked Frames: Alt + Arrow Keys: Move between linked frames. Click flow tab to jump to next/previous frame.
3. Break Link: Select frame, right-click > Break Text Link, text flows back to previous frame.
4. Unlink Frame: Delete linked frame, text flows to previous frame. Cannot have orphaned text - must have continuous flow.

4.4 Basic Paragraph Text Formatting

Frame Properties
Resizing Frames: Drag corner handles (proportional with Shift), drag side handles (stretch/squeeze). Text reflows automatically.
Frame Shape Modification: 1. Select frame with Shape Tool; 2. Individual corner nodes appear; 3. Adjust nodes to create custom frame shapes; 4. Text conforms to new shape.
Frame Margins: Text > Paragraph Text Frame > Format Frame. Set Top, Bottom, Left, Right margins ‚Äì creates padding between frame edge and text.
Frame Background: Fill frame with color, add outline. Frame and text remain separate objects.

Character Formatting in Paragraph Text All character formatting options available for Artistic Text also apply to Paragraph Text: font family, style, size, color, character effects, kerning, tracking.

Paragraph Formatting
Access Paragraph Formatting: Text Properties Docker (Ctrl+T) > Paragraph section; Property Bar when text is selected.
Alignment Options: Left, Center, Right, Full Justify, Force Justify.
Indents: First Line Indent, Left Indent, Right Indent. Set in inches, mm, points, picas.
Spacing: Character spacing, word spacing, line spacing (leading), paragraph spacing (before/after).
Tabs: View rulers, click on ruler to set tab stops. Right-click tab to change type: Left Tab, Center Tab, Right Tab, Decimal Tab. Use Tab key in text.
Bullets and Numbering: Select paragraphs, click Bullets or Numbering on Property Bar. Customize bullet character, size, position. Adjust hanging indent.

4.5 Advanced Paragraph Features

Columns
1. Select paragraph text frame
2. Go to Text > Columns
3. Set: number of columns (2-8 typical), column width (equal or variable), gutter width
4. Text automatically flows between columns
Column Types: Equal Width, Variable Width, Maintain current frame width.

Drop Caps
1. Click in paragraph
2. Go to Text > Drop Cap
3. Settings: Height (number of lines dropped 2-5 typical), distance from text, position (Dropped or Hanging)
4. Preview in real-time.

Hyphenation
1. Select text frame
2. Go to Text > Hyphenation Settings
3. Enable hyphenation
4. Set: minimum word length, minimum characters before/after hyphen, maximum consecutive hyphens
5. Manual hyphen: Ctrl+Hyphen (inserts optional hyphen).

Rules (Paragraph Rules)
1. Select paragraphs
2. Text > Paragraph Rules
3. Choose Rule Above or Rule Below
4. Set: line weight, color, style, offset from text, width relative to column or text.

Wrap Paragraph Text Around Objects
1. Place object (image, shape) overlapping paragraph text
2. Select the object
3. Click Wrap Paragraph Text on Property Bar
4. Choose wrap style: Contour, Square, Top & Bottom, Left/Right Only
5. Adjust Text Wrap Offset for spacing.

4.6 Working with Multiple Paragraph Frames

Auto-Frame Creation When importing long documents: create initial frame, import text, if overflow, CorelDRAW can auto-create pages and frames. Set in Options: Tools > Options > Text > Paragraph.

Frame Selection Techniques
- Alt + Click: Select frame behind another object
- Tab: Cycle through frames on page
- Ctrl + Click: Select frame containing text cursor

Frame Management Tools
- Align and Distribute, Group Frames, Lock Frame.

üîÑ 5. CONVERTING BETWEEN TEXT TYPES

5.1 Artistic to Paragraph Text

When to Convert: adding large amounts of text, needing columns, requiring linking, implementing drop caps.
Process: 1. Select artistic text; 2. Text > Convert to Paragraph Text (Ctrl+F8); 3. Text appears in default-sized paragraph frame; 4. Adjust frame.
Limitations: long artistic text may overflow; text on path cannot be converted directly; some effects may be lost.

5.2 Paragraph to Artistic Text

When to Convert: creating headlines from body text, applying extensive effects, fitting text to path.
Process: 1. Select paragraph text frame; 2. Text > Convert to Artistic Text (Ctrl+F8); 3. Text converts with line breaks preserved; 4. May create multiple artistic text objects if multi-line.
Important Considerations: overflow text cannot convert (must fit in one frame); multi-page text loses structure; columns removed; text frame effects lost.

5.3 Converting Both Types to Curves

Process: 1. Select text (any type); 2. Object > Convert to Curves (Ctrl+Q); 3. OR right-click > Convert to Curves.
Results: text becomes editable vector paths; each character separate curve object (if "Auto-group on Convert" is off); can ungroup to edit individual characters; font information removed.
Strategic Use: before sending to print (prevents font substitution); logo design; custom illustrations; signage. Warning: Always keep a copy of editable text before converting to curves.

‚öôÔ∏è 6. ADVANCED TEXT FORMATTING

6.1 Character Spacing Controls

Kerning (Space Between Specific Characters)
Method 1 Interactive: select text with Shape Tool, drag nodes.
Method 2 Numeric: select character(s) with Shape Tool, enter Horizontal Shift, Vertical Shift, Rotation in Property Bar.
Method 3 Range Kerning: select text with Pick Tool, in Text Properties Docker adjust Range Kerning.

Tracking (Space Across Range of Text)
1. Select text range; 2. Text Properties > Character > Range Tracking; positive increase, negative decrease.

6.2 Baseline Shift

Raise/lower characters relative to baseline: select character(s), Text Properties > Character > Baseline Shift, positive values raise, negative lower. Useful for math, trademark symbols.

6.3 OpenType Features

Accessing OpenType: select text, Text Properties Docker > Character > OpenType. Available features: Ligatures, Swash, Oldstyle Figures, Small Caps, Fractions, Ordinals (font-dependent).

6.4 Font Management

Substituting Missing Fonts: when opening files without required fonts, dialog lists missing fonts, choose substitution (temporary/permanent, match style).
Embedding Fonts in PDF: File > Publish to PDF > Settings > Objects > check "Embed fonts".
Font Matching Rules: Panose matching, default substitution in Options.

6.5 Special Characters and Symbols

Inserting Special Characters: Text > Insert Symbol Character (Ctrl+F11). Browse character sets: current font, Unicode ranges, symbol sets. Double-click to insert.
Common Shortcuts: Copyright Alt+0169, Registered Alt+0174, Trademark Alt+0153, Bullet Alt+0149, Em Dash Alt+0151, En Dash Alt+0150.
Inserting Formatting Codes: Column Break Ctrl+Shift+Enter, Frame Break Ctrl+Enter, Optional Hyphen Ctrl+Hyphen, Non-breaking Space Ctrl+Shift+Space, Non-breaking Hyphen Ctrl+Shift+Hyphen.

7.1 Fitting Text to Path (Advanced Techniques)

Creating Circular Text Draw circle, create artistic text, select both, Text > Fit Text to Path; on Property Bar place text on outside/inside, position, adjust distance.

Multi-Path Text Create compound path, fit text to compound path ‚Äì text flows across all segments.

Editing Path-Adjacent Text Double-click text to edit, use Shape Tool to adjust path.

7.2 Envelope Effects

1. Select text; 2. Effects > Envelope (Ctrl+F7); 3. Choose mode: Straight Line, Single Arc, Double Arc, Unconstrained; 4. Drag envelope nodes to distort; add/delete nodes with double-click.

7.3 3D Text Effects

Extrude: select text, Effects > Extrude, set depth, direction, lighting, color.
Bevel: Effects > Bevel, choose emboss or soft edge, set angle, depth, lighting.
Drop Shadow: Drop Shadow Tool, drag from text, adjust opacity, feathering, direction, perspective.

7.4 Text as Mask

Place image, create text over image, select both, Effects > PowerClip > Place Inside Container ‚Äì text becomes window showing image.

7.5 Perspective Effects

Select text, Effects > Add Perspective, drag corner nodes; Ctrl+click to constrain movement.

üìê 8. TYPOGRAPHY BEST PRACTICES

8.1 Font Selection Guidelines

For Headlines (Artistic Text): use bold display fonts, limit to 1-2 families, contrast with body text, ensure readability at intended size.
For Body Text (Paragraph Text): choose highly readable serif or sans-serif, 9-12 point, line spacing 120-145%, 45-75 characters per line optimal.
For Logos: unique, distinctive typefaces, custom modifications, test at small and large sizes, brand appropriateness.

8.2 Readability Principles

Line Length: ideal 45-75 characters including spaces.
Line Spacing (Leading): body text 120-145% of point size.
Alignment: left-aligned most readable, justified needs good hyphenation, centered for headlines only, right-aligned limited use.
Hierarchy: use size, weight, color, consistent heading levels, adequate contrast.

8.3 Color and Typography

High contrast black on white best; avoid low contrast; test in grayscale. Color psychology: use to emphasize, limit palette.

8.4 Professional Typesetting Tips

Punctuation: smart quotes, em dashes (‚Äî) for breaks, en dashes (‚Äì) for ranges, proper ellipsis (‚Ä¶), no double spaces.
Hyphenation: avoid more than 3 consecutive, avoid hyphenating proper names, don't hyphenate last word of paragraph.
Widows and Orphans: widow = single line at top of new column/page; orphan = single word on last line; adjust tracking/leading or edit text.
Kerning: pay attention to problem pairs (WA, To, AV); optical kerning often better than metric.

8.5 Font Formats and Licensing

Font Formats: OpenType (.otf), TrueType (.ttf), Type 1 legacy, Variable Fonts.
Licensing Considerations: Desktop License, Web License, App/Game License; free fonts: check commercial use restrictions.

‚ö†Ô∏è 9. TROUBLESHOOTING COMMON TEXT ISSUES

9.1 Font Problems

Missing Fonts: Symptom: substitute font; solution: install or choose substitutes; prevention: convert to curves.
Corrupt Fonts: Symptom: squares/gibberish; solution: reinstall, test in other apps.
Font Style Missing: Symptom: bold/italic unavailable; solution: synthetic styles sparingly; better: choose font with complete family.

9.2 Text Display Issues

Text Not Visible: check color (white on white), layer order, outline/no fill, zoom.
Text Pixelated: View > Enhanced mode, check zoom, verify not converted to bitmap.
Text Overflow Hidden: look for red flow indicator, resize frame or link, check frame properties.

9.3 Printing and Export Problems

Text Reflows in PDF: embed fonts, convert to curves, check PDF settings.
Missing Fonts in Print: printer missing font; embed/convert, provide fonts to print shop.
Wrong Font in Export: export settings may substitute; use PDF with embedded fonts; convert to curves before export.

9.4 Performance Issues

Slow with Large Documents: split files, use linked frames efficiently, purge undo list (Edit > Undo > Clear Undo), convert completed sections to curves.
Text Heavy Files: optimize images, use styles consistently, avoid excessive effects, save regularly.
