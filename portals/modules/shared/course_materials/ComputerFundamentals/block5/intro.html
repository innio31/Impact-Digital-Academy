üìò CHAPTER 1: INTRODUCTION TO GRAPHIC DESIGN

What is Graphic Design?

Graphic design is the art and practice of planning and projecting ideas and experiences with visual and textual content. It is a form of visual communication that combines images, words, and ideas to convey information to an audience, especially to produce a specific effect.

In the context of desktop publishing and secretarial studies, graphic design skills enable professionals to:

  • Create visually appealing business documents
  • Design effective marketing materials
  • Enhance corporate communications
  • Maintain brand consistency across all materials
  • Communicate complex information clearly and efficiently

The Role of Graphic Design in Business

Effective graphic design in business:

  • Establishes professional credibility
  • Enhances brand recognition
  • Improves communication efficiency
  • Increases audience engagement
  • Supports marketing and sales objectives
  • Creates emotional connections with audiences

üîπ CHAPTER 2: DESIGN ELEMENTS

Design elements are the basic components or building blocks used to create any visual composition. Think of them as the "ingredients" in a recipe. Every design, regardless of complexity, is constructed from these fundamental elements.

2.1 LINE

Definition

A line is a mark connecting two points. It is the most basic element of design and can be defined as a moving dot. Lines have length, direction, and position.

Types of Lines

Line Type Characteristics Psychological Effect Common Uses
Straight Lines Direct, precise, predictable Stability, order, strength Borders, grids, technical drawings
Curved Lines Gentle, flowing, organic Grace, comfort, movement Logos, illustrations, decorative elements
Horizontal Lines Parallel to horizon Calm, stability, rest Headers, dividers, grounding elements
Vertical Lines Perpendicular to horizon Strength, dignity, growth Columns, pillars, upward movement
Diagonal Lines Slanted, dynamic Action, tension, energy Movement indicators, dynamic compositions
Zigzag Lines Angular, sharp changes Excitement, chaos, danger Attention-grabbing elements
Implied Lines Suggested, not continuous Connection, flow Visual pathways, eye movement guides

Practical Applications in Desktop Publishing

Example 1: Business Report
- Use horizontal lines to separate sections
- Use vertical lines to create columns
- Use thin rules under headings for emphasis

Example 2: Brochure Design
- Use diagonal lines to create dynamic layouts
- Use curved lines to guide the eye around the page
- Use implied lines through alignment of elements

Technical Considerations

  • Line weight (thickness) affects visual hierarchy
  • Line style (solid, dashed, dotted) conveys different meanings
  • Line color should contrast sufficiently with background
  • In CorelDRAW, use the Freehand Tool, Bezier Tool, or Shape Tool to create lines

2.2 SHAPE

Definition

A shape is a defined area created by lines, color changes, or texture differences. Shapes are two-dimensional and have height and width.

Categories of Shapes

Category Description Examples Psychological Impact
Geometric Shapes Mathematical, precise, regular Squares, circles, triangles, rectangles Order, stability, efficiency, professionalism
Organic Shapes Free-form, natural, irregular Leaves, clouds, puddles, abstract blobs Natural, friendly, creative, relaxed
Abstract Shapes Simplified versions of organic shapes Stick figures, icons, symbols Universal recognition, communication efficiency

Shape Meanings and Associations

Circles and Ovals:
- Represent infinity, protection, wholeness
- Suggest community, relationships, unity
- Often used in logos for harmony-focused companies
- Directs eye in a circular path around the composition

Squares and Rectangles:
- Represent stability, honesty, security
- Suggest professionalism, reliability
- Most common shape for printed materials (pages, photos)
- Create structure and organization

Triangles:
- Represent action, tension, energy
- Suggest direction (pointing upward = aspiration, downward = stability)
- Create dynamic compositions
- Draw attention to the point

Practical Applications:

Business Card Design:
- Rectangle shape for the card itself
- Circular logo for approachability
- Square photo for stability

Flyer Design:
- Use shapes as containers for text
- Create visual interest with overlapping shapes
- Use shapes to direct attention to key information

Technical Considerations in CorelDRAW

  • Rectangle Tool (F6) for perfect rectangles and squares (hold Ctrl)
  • Ellipse Tool (F7) for circles and ellipses
  • Polygon Tool for multi-sided shapes
  • Shape Tool (F10) to modify shapes
  • Combine shapes using Weld, Trim, Intersect functions

2.3 COLOR

Definition

Color is the element of design that results from light reflecting off objects. It is perhaps the most powerful and emotional design element, capable of conveying mood, attracting attention, and organizing information.

The Color Wheel

The color wheel is a visual representation of colors arranged according to their chromatic relationship. It is fundamental to understanding color relationships.

Primary Colors:
- Red, Blue, Yellow
- Cannot be created by mixing other colors
- The foundation of all other colors

Secondary Colors:
- Green (Blue + Yellow)
- Orange (Red + Yellow)
- Purple (Red + Blue)
- Created by mixing two primary colors

Tertiary Colors:
- Red-Orange, Yellow-Orange, Yellow-Green, Blue-Green, Blue-Purple, Red-Purple
- Created by mixing a primary with an adjacent secondary

Color Properties

Property Definition Example
Hue The pure color itself (the name of the color) Red, Blue, Green
Saturation (Chroma) The purity or intensity of a color Vivid red vs. dull gray-red
Value (Brightness) The lightness or darkness of a color Light blue vs. dark blue

Color Temperature

Warm Colors:
- Reds, oranges, yellows
- Advance toward the viewer
- Energetic, exciting, stimulating
- Use for attention-grabbing elements

Cool Colors:
- Blues, greens, purples
- Recede from the viewer
- Calming, professional, trustworthy
- Use for backgrounds and professional communications

Practical Color Applications

For Professional Documents:

‚öñÔ∏è CHAPTER 3: DESIGN PRINCIPLES

Design principles are guidelines for organizing the design elements effectively. They help create compositions that are visually appealing, functional, and communicative.

3.1 BALANCE

Definition

Balance refers to the distribution of visual weight in a composition. It creates stability and structure in a design.

Types of Balance

Balance Type Description Example Best Used For
Symmetrical Balance Elements mirror each other on either side of a central axis Formal invitations, certificates Traditional, formal, stable designs
Asymmetrical Balance Balance achieved through contrast of different elements with equal visual weight Modern brochures, magazines Dynamic, interesting, contemporary designs
Radial Balance Elements radiate from a central point Mandalas, some logos, CD labels Drawing attention to center, decorative designs
Mosaic/Crystallographic Balance Chaos balanced by overall pattern Jackson Pollock paintings, all-over patterns Backgrounds, textile designs

Achieving Visual Weight

Visual weight is determined by:

Element Characteristic Heavier (More Weight) Lighter (Less Weight)
Size Large objects Small objects
Color Bright, warm colors Muted, cool colors
Value Dark values Light values
Texture High texture/detail Smooth, minimal texture
Position Isolated elements Clustered elements
Shape Irregular shapes Regular shapes

Practical Examples

Symmetrical Balance Example - Formal Letter:
- Company logo centered at top
- Equal margins left and right
- Recipient address block centered or left-aligned with equal white space

Asymmetrical Balance Example - Brochure Cover:
- Large image on right side (heavy)
- Small logo and bold headline on left (light but attention-grabbing)
- Balance achieved through color and typography weight

Tips for Achieving Balance

  1. Step back and squint at your design
  2. Look for areas that feel too heavy or too light
  3. Use the "see-saw" mental model
  4. Check balance in grayscale to remove color influence
  5. Consider both horizontal and vertical balance

3.2 CONTRAST

Definition

Contrast is the arrangement of opposite elements to create visual interest, excitement, and emphasis. It helps organize information and guide the viewer's eye.

Types of Contrast

Contrast Type Examples Effect
Size Contrast Large vs. small elements Creates hierarchy, drama
Color Contrast Complementary colors, light vs. dark Attracts attention, separates elements
Value Contrast Light vs. dark Creates depth, readability
Shape Contrast Geometric vs. organic Visual interest, tension
Texture Contrast Smooth vs. rough Tactile interest, separation
Typography Contrast Serif vs. sans-serif, bold vs. light Hierarchy, personality
Direction Contrast Horizontal vs. vertical Dynamic movement

The Importance of Contrast

Without contrast, designs appear:
- Flat and uninteresting
- Difficult to navigate
- Unprofessional
- Hard to read

With effective contrast:
- Information hierarchy is clear
- Visual interest is maintained
- Key elements stand out
- Readability improves

Contrast in Readability

Text and Background Contrast Requirements:
- WCAG (Web Content Accessibility Guidelines) recommends:
- Normal text: 4.5:1 contrast ratio minimum
- Large text: 3:1 contrast ratio minimum

Examples of Good Contrast:
- Black text on white background (highest readability)
- Dark colored text on light colored background
- White text on dark colored background

Examples of Poor Contrast:
- Light gray text on white background
- Red text on green background
- Yellow text on white background

Practical Applications

Headlines and Body Text:
- Use large, bold, contrasting font for headlines
- Use smaller, lighter, simpler font for body text
- Ensure sufficient contrast between text and background

Call-to-Action Buttons:
- Use high-contrast colors that stand out
- Make buttons larger than surrounding elements
- Use contrasting shapes (rounded buttons on square design)

Tips for Effective Contrast

  1. Use contrast to create a focal point
  2. Don't use contrast for everything (it becomes noise)
  3. Test contrast by viewing in grayscale
  4. Ensure sufficient contrast for accessibility
  5. Use contrast to establish hierarchy

3.3 ALIGNMENT

Definition

Alignment is the placement of elements so they line up along common edges or axes. It creates order, organization, and visual connections between elements.

Types of Alignment

Alignment Type Description Best Used For
Left Alignment Text/elements align on the left edge Body text, Western languages
Right Alignment Text/elements align on the right edge Captions, dates, pull quotes
Center Alignment Elements centered on a central axis Headlines, invitations, titles
Justified Alignment Text aligns on both left and right Newspapers, magazines, books
Edge Alignment Elements align to page edges Modern layouts, minimal designs
Optical Alignment Visual alignment adjusting for shape quirks Logos, professional typography

The Power of Alignment

Good alignment:
- Creates visual connections between elements
- Reduces visual clutter
- Creates a clean, professional appearance
- Guides the viewer through the design
- Creates visual rhythm

Poor alignment:
- Creates visual chaos
- Looks unprofessional and sloppy
- Confuses the viewer
- Breaks visual connections

Grid Systems

A grid is a system for organizing content using columns, rows, and margins.

Common Grid Types:

Grid Type Description Applications
Manuscript Grid Single column of text Books, reports, essays
Column Grid Multiple columns Newspapers, magazines, brochures
Modular Grid Columns and rows Forms, tables, complex layouts
Hierarchical Grid Based on content needs Web design, posters

Benefits of Using Grids:
- Creates consistency across multiple pages
- Speeds up the design process
- Ensures proper alignment
- Creates visual harmony
- Makes responsive design easier

Practical Applications

Business Document:
- Align all text to a consistent left margin
- Align page numbers consistently
- Align headings with body text or use consistent indentation
- Align graphics with text columns

Multi-page Document:
- Use consistent margins throughout
- Align elements consistently across pages
- Use a grid to maintain consistency
- Align headers and footers

Common Alignment Mistakes

  1. Using multiple alignments on one page without purpose
  2. Centering body text
  3. Ignoring the optical alignment of curved letters
  4. Not using enough white space around aligned elements

Tips for Perfect Alignment

  1. Use guides and grids in your software
  2. Zoom in to check precise alignment
  3. Group related elements for consistent movement
  4. Align to a consistent baseline grid for text
  5. Break alignment purposefully for emphasis (then restore it)

3.4 REPETITION

Definition

Repetition involves repeating visual elements throughout a design to create consistency, unity, and visual interest. It strengthens the design by tying together individual parts.

What Can Be Repeated

Element Type Examples of Repetition
Color Brand colors throughout a document
Shape Using same shape style for all graphics
Typography Consistent heading and body fonts
Line Weight Consistent border thickness
Texture Same texture on related elements
Spatial Relationships Consistent spacing and margins
Format Consistent document structure

Repetition vs. Pattern

Repetition: The use of the same element multiple times
- Example: Using the same heading style throughout a report

Pattern: A specific arrangement of repeated elements
- Example: A geometric pattern created from repeated shapes

Benefits of Repetition

Consistency:
- Creates a cohesive look
- Reinforces brand identity
- Makes multi-page documents feel unified

Navigation:
- Helps readers understand document structure
- Creates predictable layouts
- Guides readers to find information

Professionalism:
- Shows attention to detail
- Demonstrates design skill
- Creates polished appearance

Practical Applications

Brand Identity:
- Repeat logo placement consistently
- Use consistent color palette throughout all materials
- Use consistent typography across all communications

Reports and Documents:
- Use same heading styles throughout
- Repeat page numbers in same position
- Use consistent bullet styles
- Repeat footer elements on every page

Marketing Materials:
- Repeat brand elements across all pieces
- Use consistent image treatment
- Repeat call-to-action styling

Examples of Repetition in Business Documents

Corporate Report:
- Same header style on every page
- Same footer with consistent information
- Same treatment for all section headings
- Same bullet style for all lists

Brochure Series:
- Same color scheme across all brochures
- Same typography choices
- Same layout structure
- Same image treatment

Tips for Effective Repetition

  1. Establish a clear system before starting
  2. Document your choices (create a style guide)
  3. Be consistent but not boring
  4. Use repetition to create rhythm
  5. Occasionally break repetition for emphasis

3.5 HIERARCHY

Definition

Hierarchy is the organization of elements to show their order of importance. It guides the viewer through the content, telling them what to look at first, second, and third.

Creating Visual Hierarchy

Method How It Works Example
Size Larger elements attract attention first Large headline, smaller subhead, smallest body text
Color Bright or contrasting colors attract attention Red call-to-action button
Position Top and center are primary positions Logo at top of page
Weight Bold elements attract attention Bold headings, light body text
Space More space around an element emphasizes it Isolated logo on cover
Texture/Pattern Textured elements stand out from smooth Patterned background for highlight
Style Distinctive styling creates emphasis Drop caps, pull quotes

The Visual Hierarchy Pyramid

Primary Message ‚ñ≤ ‚îÇ Secondary Elements ‚ñ≤ ‚îÇ Supporting Details ‚ñ≤ ‚îÇ Background Elements

Level 1 - Primary Message:
- Main headline
- Primary image
- Logo
- Call-to-action

Level 2 - Secondary Elements:
- Subheadings
- Section titles
- Navigation elements
- Key supporting images

Level 3 - Supporting Details:
- Body text
- Captions
- Data
- Secondary images

Level 4 - Background Elements:
- Page numbers
- Footer information
- Background textures
- Decorative elements

Reading Patterns

Understanding how people read helps establish hierarchy:

Z-Pattern (for simple layouts):
1. Top left (primary)
2. Top right (secondary)
3. Bottom left (tertiary)
4. Bottom right (call-to-action)

F-Pattern (for text-heavy layouts):
1. Top headline
2. First few lines of text
3. Left-aligned headings down the page

Practical Applications

Business Letter:
- Level 1: Company logo, date, recipient name
- Level 2: Subject line, salutation
- Level 3: Body paragraphs
- Level 4: Signature block, enclosures

Brochure Cover:
- Level 1: Main headline
- Level 2: Primary image, subheadline
- Level 3: Logo, date/location
- Level 4: Background color/texture

Report Cover:
- Level 1: Report title
- Level 2: Company logo, report subtitle
- Level 3: Date, author, department
- Level 4: Background elements

Hierarchy Mistakes to Avoid

  1. Making everything important (nothing stands out)
  2. Conflicting hierarchy cues (large but light gray text)
  3. Ignoring natural reading patterns
  4. Poor contrast in important elements
  5. Inconsistent hierarchy across pages

Tips for Effective Hierarchy

  1. Decide on the single most important element
  2. Use size as the primary hierarchy tool
  3. Support size with color and weight
  4. Test hierarchy by squinting
  5. Get feedback on what viewers see first

4.1 COLOR MODES: CMYK vs. RGB

Understanding color modes is essential for ensuring your designs appear correctly in their final format.

RGB (Red, Green, Blue)

Characteristic Description
Full Name Red, Green, Blue
Color Model Additive color model
How It Works Colors are created by adding light; black is absence of light, white is all colors combined
Gamut Wider color range (more colors than CMYK)
Primary Use Digital displays (monitors, TVs, phones, projectors)
File Formats JPEG, PNG, GIF, SVG (for web)
When to Use Web design, social media graphics, presentations, videos, any screen-only output

Technical Details:
- Each channel has 256 values (0-255)
- 0,0,0 = Black
- 255,255,255 = White
- 16.7 million possible colors

CMYK (Cyan, Magenta, Yellow, Key/Black)

Characteristic Description
Full Name Cyan, Magenta, Yellow, Key (Black)
Color Model Subtractive color model
How It Works Colors are created by subtracting light using ink on paper; white is absence of ink, black is all inks combined
Gamut Narrower color range (limited by ink and paper)
Primary Use Commercial printing (brochures, business cards, flyers, magazines)
File Formats PDF, TIFF, AI, CDR (for print)
When to Use Any project that will be professionally printed

Technical Details:
- Each channel has percentages (0-100%)
- 0,0,0,0 = White (paper color)
- 100,100,100,100 = Rich black (but rarely used)
- Approximately 1 million printable colors

Conversion Considerations

RGB to CMYK Conversion Issues:
- Bright RGB colors often become dull in CMYK
- Neon and fluorescent colors cannot be reproduced in CMYK
- Some blues and purples shift significantly
- Always convert before finalizing print designs

Best Practices:
1. Design in the mode you'll output in
2. If designing for both, design in RGB first, then convert copy for print
3. Use Pantone Matching System (PMS) for exact color matching
4. Proof your designs before mass printing

Spot Colors (Pantone)

Characteristic Description
Definition Pre-mixed ink colors, not created by CMYK combination
Advantages Exact color matching, consistent results, special effects (metallic, fluorescent)
Disadvantages More expensive, additional printing plate required
When to Use Brand colors, logos, high-end printing

4.2 COLOR HARMONIES

Color harmonies are combinations of colors that are pleasing to the eye, based on their positions on the color wheel.

Monochromatic Harmony

Aspect Description
Definition Uses variations of a single hue (different tints, shades, tones)
Color Wheel Relationship One hue only
Characteristics Harmonious, cohesive, elegant, subtle
Advantages Always works, creates unity, professional appearance
Disadvantages Can be boring without texture or contrast
Best Uses Corporate materials, minimalist designs, elegant invitations

Creating Monochromatic Schemes:
- Tint: Add white to lighten
- Shade: Add black to darken
- Tone: Add gray to mute

Analogous Harmony

Aspect Description
Definition Uses colors adjacent to each other on the color wheel
Color Wheel Relationship 2-4 colors next to each other
Characteristics Harmonious, serene, comfortable, natural
Advantages Rich color schemes, pleasing to eye, easy to create
Disadvantages Low contrast, can lack energy
Best Uses Nature-themed designs, backgrounds, calming materials

Example Schemes:
- Blue, Blue-Green, Green
- Red, Red-Orange, Orange
- Yellow, Yellow-Green, Green

Complementary Harmony

Aspect Description
Definition Uses colors opposite each other on the color wheel
Color Wheel Relationship Directly opposite
Characteristics High contrast, vibrant, energetic, exciting
Advantages Maximum visibility, attention-grabbing, dynamic
Disadvantages Can be jarring if overused, difficult to balance
Best Uses Calls-to-action, logos, attention-grabbing elements

Common Complementary Pairs:
- Red and Green
- Blue and Orange
- Yellow and Purple

Split Complementary Harmony

Aspect Description
Definition One color plus the two colors adjacent to its complement
Characteristics High contrast like complementary but less tension, more versatile
Best Uses Versatile designs requiring contrast without harshness

Example:
- Base: Blue
- Complement's neighbors: Yellow-Orange and Red-Orange

Triadic Harmony

Aspect Description
Definition Three colors equally spaced on the color wheel
Characteristics Vibrant, balanced, rich, even contrast
Advantages Strong visual interest, balanced energy
Disadvantages Can be overwhelming if all colors are saturated
Best Uses Bold designs, children's materials, vibrant branding

Primary Triad:
- Red, Blue, Yellow

Secondary Triad:
- Green, Orange, Purple

Tetradic (Double Complementary) Harmony

Aspect Description
Definition Two complementary color pairs
Characteristics Rich, complex, full of variety
Advantages Most color variety, rich schemes
Disadvantages Difficult to balance, can be chaotic
Best Uses Complex designs, illustrations, advanced projects

The 60-30-10 Rule

A practical rule for applying color harmonies:

Percentage Usage Example
60% Dominant color - main background Walls, large areas
30% Secondary color - supporting elements Furniture, key sections
10% Accent color - highlights Accessories, call-to-action

Color Temperature in Harmonies

Warm Harmonies:
- Reds, oranges, yellows
- Advance, energize, excite
- Use for foreground elements

Cool Harmonies:
- Blues, greens, purples
- Recede, calm, professional
- Use for backgrounds

Practical Application Steps

  1. Start with Purpose: What feeling should the design convey?
  2. Choose Base Color: Usually brand color or main message color
  3. Select Harmony Type: Based on desired effect
  4. Apply 60-30-10 Rule: Ensure balanced application
  5. Test in Context: View in different sizes and on different media

5.1 SERIF vs. SANS-SERIF FONTS

Understanding Type Classifications

Serif Fonts

Aspect Description
Definition Fonts with small lines or strokes attached to the ends of larger strokes
Origin Derived from Roman stone carving; serifs helped finish strokes cleanly
Characteristics Traditional, formal, authoritative, trustworthy, readable in long text
Readability Excellent for long passages of text (books, newspapers)
Common Examples Times New Roman, Garamond, Georgia, Baskerville, Caslon

Sans-Serif Fonts

Aspect Description
Definition Fonts without serifs ("sans" = French for "without")
Origin Developed in the 19th century, popularized in the 20th century
Characteristics Modern, clean, informal, approachable, simple
Readability Excellent for headlines, screens, and short text
Common Examples Arial, Helvetica, Verdana, Calibri, Futura, Open Sans

Comparison Guide

Use Case Serif Recommendation Sans-Serif Recommendation
Long body text (print) ‚úì Excellent Acceptable
Long body text (screen) Acceptable ‚úì Better (lower resolution)

Formal documents ‚úì Excellent Acceptable

High resolution ‚úì Excellent ‚úì Excellent

Other Type Classifications

Script Fonts:
- Mimic handwriting or calligraphy
- Elegant, personal, formal
- Use sparingly for invitations, logos
- Difficult to read in long passages

Display/Decorative Fonts:
- Highly stylized, unique designs
- Strong personality, attention-grabbing
- Use for headlines only, never body text
- One per design maximum

Monospace Fonts:
- Each character occupies same width
- Technical, coding, typewriter feel
- Use for code, technical documents, retro designs

5.2 FONT PAIRING

Principles of Font Pairing

Contrast is Key: Successful font pairings usually combine different but complementary typefaces. Similar fonts often clash.

The Golden Rule: Pair a serif with a sans-serif for best results.

Successful Pairing Strategies

Strategy Description Example
1. Serif + Sans-Serif Classic, reliable, creates clear hierarchy Garamond (headline) + Helvetica (body)
2. Same Family Different weights/styles of same typeface Light, regular, bold of Open Sans
3. Superfamily Type families designed with both serif and sans-serif versions Scala & Scala Sans
4. Similar Structure Fonts with similar proportions but different personalities Both have large x-height
5. Historical Connection Fonts from same time period Baskerville (18th) + Didot (18th)

Font Mood Matching

Ensure font personalities work together:

Mood Serif Options Sans-Serif Options
Professional/Formal Garamond, Times, Caslon Frutiger, Myriad
Friendly/Approachable Georgia, Chaparral Arial, Calibri, Verdana
Modern/Clean Bodoni, Didot Helvetica, Futura
Creative/Artistic Caslon, Adobe Jenson Gotham, Proxima Nova
Traditional/Trustworthy Garamond, Minion Univers, Frutiger

Common Pairing Examples

Headline Font Body Font Best For

Font Pairing Mistakes

  1. Too Similar: Two serifs that compete
  2. Too Different: Display font + highly decorative script
  3. Too Many Fonts: More than three creates chaos
  4. Wrong Context: Comic Sans in legal document
  5. Poor Contrast: Both medium weight, similar size

5.3 TYPOGRAPHY TERMINOLOGY

Essential Terms

Leading (pronounced "ledding")

Aspect Description
Definition Vertical space between lines of text
Measurement Usually measured in points (pt)
Rule of Thumb 120-145% of font size (e.g., 10pt type, 12-14.5pt leading)
Too Tight Lines touch, difficult to read, crowded
Too Loose Lines disconnect, reading flow broken

Optimal Leading by Use:

Text Type Leading Recommendation
Body text 120-145% of type size
Headlines 100-120% of type size (tighter)
Captions 130-150% of type size (looser)
All caps 100-110% of type size

Kerning

Aspect Description
Definition Space between individual character pairs
Purpose Creates visually even spacing despite character shapes
Problem Pairs AV, WA, To, Tr, Ya (need adjustment)
Good Kerning Invisible; space appears even to the eye
Tools Manual adjustment in professional software

Common Kerning Pairs to Check:
- A V (too loose usually)
- L Y (too loose)
- T o (can appear too tight)
- F o (can appear too tight)
- r n (can look like "m")

Tracking (Letter-spacing)

Aspect Description
Definition Uniform space across a range of characters
Difference from Kerning Tracking affects all letters equally; kerning adjusts pairs
Normal Text 0 tracking (default)
All Caps Often needs + tracking (5-50)
Headlines May need - tracking (tighter)
Small Text May need + tracking (looser)

Tracking Guidelines:

Text Situation Tracking Adjustment
Body text (upper/lowercase) 0 (default)
Body text (all caps) +10 to +50
Headlines (upper/lowercase) -10 to -20
Headlines (all caps) 0 to +20
Very small text (<10pt)< /td> +5 to +15

Additional Typography Terms

x-height: Height of lowercase letters (excluding ascenders/descenders)
- Large x-height = better readability at small sizes
- Small x-height = more elegant, needs larger size

Baseline: Invisible line where letters sit

Ascender: Part of letter extending above x-height (b, d, f, h)

Descender: Part of letter extending below baseline (g, j, p, q)

Counter: Enclosed space in letters (o, p, d)

Ligature: Combined characters (Ô¨Å, Ô¨Ä, Ô¨Ç)

5.4 PRACTICAL TYPOGRAPHY RULES

For Professional Documents

  1. Limit Font Families: Maximum 2-3 per document
  2. Establish Hierarchy:
    - Heading 1: Largest, boldest
    - Heading 2: Smaller than H1
    - Body text: 10-12pt for print, 14-16px for web
    - Captions: Smaller than body text
  3. Line Length (Measure):
    - Optimal: 45-75 characters per line
    - Too short: Choppy reading
    - Too long: Difficult to track next line
  4. Paragraph Spacing:
    - Use space after paragraphs, not indents for business
    - Space after = 50-100% of line height
  5. Alignment:
    - Left align body text for easiest reading
    - Center headlines (sparingly)
    - Avoid justified text with poor hyphenation
    - Avoid right-aligned body text
  6. Widows and Orphans:
    - Widow: Single line at top of page
    - Orphan: Single word on last line
    - Avoid both through editing or manual adjustment
  7. Emphasis:
    - Use bold for strong emphasis
    - Use italic for gentle emphasis
    - Avoid underlining (reserved for links)
    - Avoid ALL CAPS for long passages

Typography for Accessibility

  • Minimum 12pt for body text in print
  • Minimum 16px for body text on web
  • Strong contrast (4.5:1 minimum)
  • Avoid light text on light backgrounds
  • Avoid small text on textured backgrounds

6.1 RASTER vs. VECTOR GRAPHICS

Understanding the fundamental difference between raster and vector graphics is essential for professional design work.

RASTER GRAPHICS

Aspect Description
Definition Images composed of a grid of individual pixels
Also Known As Bitmap images
Composition Each pixel contains color information
Scalability Limited - enlarging causes pixelation and quality loss
File Size Increases with resolution and dimensions
Editing Pixel-level editing, photo manipulation

Best Uses for Raster:
- Photographs
- Complex images with gradients
- Web images
- Digital art
- Scanned documents

Limitations:
- Not infinitely scalable
- Large file sizes at high resolution
- Loss of quality with transformations

VECTOR GRAPHICS

Aspect Description
Definition Images composed of mathematical paths and curves
Also Known As Object-oriented graphics
Composition Points, lines, curves, and shapes defined mathematically
Scalability Infinite - can be scaled to any size without quality loss
File Size Generally smaller, based on complexity not dimensions
Editing Object-level editing, easy color and shape changes

Best Uses for Vector:
- Logos and branding
- Illustrations
- Typography
- Technical drawings
- Icons
- Any design needing scaling to multiple sizes

Limitations:
- Not suitable for photorealistic images
- Limited gradient and texture capabilities
- Requires specialized software

Comparison Table

Feature Raster Vector
Composition Pixels Mathematical paths
Scalability Limited Infinite
File Size Based on dimensions Based on complexity
Photorealism Excellent Poor
Logo Suitability Poor Excellent
Software Photoshop, GIMP CorelDRAW, Illustrator
Common Formats JPEG, PNG, TIFF, GIF AI, CDR, SVG, EPS

6.4 RESOLUTION: DPI vs. PPI

Understanding Resolution

PPI (Pixels Per Inch)

Aspect Description
Definition Number of pixels in one inch of a digital image
Applies To Digital images, screens, cameras
Formula PPI = Pixel dimensions / Physical dimensions
Example 3000px wide / 10 inches wide = 300 PPI

DPI (Dots Per Inch)

Aspect Description
Definition Number of ink dots printed per inch
Applies To Printed output, printers
Determined By Printer capability and settings
Example 600 DPI printer places 600 dots per inch

Resolution Requirements by Output

Output Type Recommended Resolution Notes
Web/Screen 72-96 PPI Screen resolution only
Office Printing 150-200 PPI Standard laser/inkjet
Professional Print 300 PPI Magazines, brochures
High-Quality Print 300-600 PPI Art books, photography
Large Format 100-150 PPI Billboards, banners*

*Large format viewed from distance requires lower PPI

Calculating Required Resolution

Formula: Required PPI √ó Output Size (inches) = Pixel Dimensions Needed

Example - Full Page Brochure (8.5" √ó 11"):
- 300 PPI √ó 8.5" = 2550 pixels wide
- 300 PPI √ó 11" = 3300 pixels tall
- Required image size: 2550 √ó 3300 pixels

Common Mistakes and Solutions

Mistake Problem Solution
Using 72 PPI for print Pixelated output Start with high-res images
Upsampling low-res images Blurry, soft results Reshoot or rescan
Confusing DPI with PPI Wrong file preparation Understand the difference
Ignoring viewing distance Wrong resolution choice Consider final viewing conditions

Resolution Checklist for Print Projects

  • ‚òê All images at least 300 PPI at final size
  • ‚òê Check resolution after any scaling
  • ‚òê Use high-res source images
  • ‚òê Save at full resolution until final
  • ‚òê Convert to appropriate format for output
