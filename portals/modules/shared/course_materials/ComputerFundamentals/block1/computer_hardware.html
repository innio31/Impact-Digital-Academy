üîπ 1. Introduction to Computer Hardware

What is Computer Hardware?

Computer hardware refers to the physical, tangible components that make up a computer system. Unlike software, which consists of instructions and data that tell the hardware what to do, hardware can be seen and touched.

The Von Neumann Architecture

Most modern computers are based on the Von Neumann architecture, which consists of four main components:

  • Processing Unit (CPU): Executes instructions
  • Memory Unit (RAM): Temporarily stores data and instructions
  • Storage: Permanently saves data
  • Input/Output Devices: Facilitates communication with the outside world

The Information Processing Cycle

  1. Input: Data enters the computer
  2. Processing: Data is manipulated/calculated
  3. Storage: Data is saved temporarily or permanently
  4. Output: Results are presented to the user

2.1 What is the CPU?

The Central Processing Unit, often called the "brain" of the computer, is responsible for executing instructions from programs. It performs all arithmetic, logical, control, and input/output operations.

2.2 Components of CPU

a) Control Unit (CU)

  • Function: Directs the operation of the processor
  • Responsibilities: Fetches instructions from memory; decodes instructions; controls data flow; generates timing and control signals

b) Arithmetic Logic Unit (ALU)

  • Function: Performs all arithmetic and logical operations
  • Arithmetic: addition, subtraction, multiplication, division
  • Logical: comparisons (greater than, less than, equal to), AND, OR, NOT
  • Bit manipulation: shifting, rotating bits

c) Registers

Definition: Small, high-speed storage locations within the CPU

  • MAR: Holds address of memory location to access
  • MBR/MDR: Holds data being transferred
  • PC (Program Counter): Holds address of next instruction
  • IR (Instruction Register): Holds current instruction
  • ACC (Accumulator): Stores intermediate arithmetic results

2.3 CPU Performance Factors

Clock speed: cycles/sec (GHz), 2-5 GHz modern
Core count: single, dual, quad, hexa, octa, many‚Äëcore
Cache: L1 (fastest), L2, L3 (shared)

2.4 CPU Technologies

  • Hyper-Threading / SMT: one physical core ‚Üí two logical cores
  • Turbo Boost / Precision Boost: automatic overclock when needed
  • Thermal Design Power (TDP): max heat (watts), affects cooling

  • Air cooling: heat sink + fan
  • Liquid cooling: AIO or custom loop
  • Passive cooling: no fans, only heat sinks

üß† 3. Memory (RAM)

3.1 What is RAM?

Random Access Memory is the computer's primary volatile memory where data and instructions are temporarily stored for quick access by the CPU.

3.2 Key Characteristics

  • Volatility: data lost when power off
  • Speed: faster than storage, slower than CPU cache (nanoseconds)
  • Random access: any location directly accessible

3.3 Types of RAM

  • DRAM (Dynamic RAM): needs refresh, main memory
  • SRAM (Static RAM): no refresh, cache memory
  • SDRAM: synchronized with CPU clock
  • DDR SDRAM generations: DDR1 to DDR5 (speeds, voltage table)
Generation Transfer rate Voltage Common speeds
DDR1 200-400 MT/s 2.5V obsolete
DDR2 400-800 MT/s 1.8V legacy
DDR3 800-2133 MT/s 1.5V older computers
DDR4 1600-3200 MT/s 1.2V current standard
DDR5 4800-6400 MT/s 1.1V latest

3.4 RAM Modules

Form factors: DIMM (desktop), SO-DIMM (laptop). Packaging: single/dual sided, single/dual rank.

3.5 RAM Performance Factors

  • Capacity: 8GB min, 16GB rec, 32-128GB pro, servers TB
  • Speed (MT/s): e.g., DDR4-3200
  • Latency (CAS): CL numbers, e.g., 16-18-18-36
  • Multi-channel: dual/triple/quad increases bandwidth

3.6 Virtual Memory

Using hard drive space as ‚Äúfake‚Äù RAM (page file / swap). Much slower than physical RAM.

3.7 How RAM Works with CPU

  1. CPU requests address
  2. Memory controller sends address to RAM
  3. RAM retrieves data
  4. Data travels to CPU
  5. CPU stores in cache
