üìò 1. Introduction to Excel Formulas

What is a Formula?

A formula in Excel is an expression that calculates values. Every formula begins with an equals sign (=).

Basic structure: = operand operator operand ‚Üí e.g., = 5 + 3

What is a Function?

A function is a predefined formula that uses arguments. = FUNCTION_NAME(argument1, argument2)

Example: =SUM(A1:A10)

The Formula Bar

Located above the worksheet; displays active cell‚Äôs content and allows editing.

Order of Operations (PEMDAS)

  • Parentheses
  • Exponents
  • Multiplication
  • Division
  • Addition
  • Subtraction

= 5 + 3 * 2 ‚Üí 11 (multiplication first)
= (5 + 3) * 2 ‚Üí 16 (parentheses first)

‚ûï 2. Creating Basic Mathematical Formulas

Addition (+)

= number1 + number2 + ...

Formula Description Result
=5+3 adds 5 and 3 8
=A1+B1 adds values in A1 and B1 depends
=A1+B1+C1 adds three cells depends

Practical ‚Äì monthly expenses: =A1+A2+A3+A4 (Rent, Utilities, Groceries, Transport)

Subtraction (-)

= number1 - number2

Example Result
=10-4 6
=B1-B2 Revenue - Costs

Multiplication (*)

= number1 * number2 ‚Üí =C1*D1 (Unit price * quantity)

Division (/)

= number1 / number2 ‚Üí =E1/F1 (Total cost / units) ‚Üí avoid division by zero

Combined operations

=G1 + (G1*G2) ‚Üí subtotal + tax
=(H1+H2+H3)/3 ‚Üí average of three scores
=I1 * I2 ‚Üí commission = sales * rate

‚öôÔ∏è 3. Essential Excel Functions

SUM

=SUM(number1, [number2], ...) ‚Äì adds numbers, ranges, or a mix.

  • =SUM(A1:A10) , =SUM(A1:A5, C1:C5, 100)

AVERAGE

=AVERAGE(number1, ...) ‚Äì arithmetic mean. Ignores blanks but includes zero.

MIN & MAX

=MIN(range) ‚Äì smallest; =MAX(range) ‚Äì largest.

COUNT

=COUNT(value1, ...) ‚Äì counts cells containing numbers.

Function Counts Ignores
COUNT numbers only text, blanks, errors
COUNTA any non‚Äëempty blanks
COUNTBLANK empty cells filled cells

Examples: =COUNT(B2:B30) ‚Üí number of tests taken; =COUNTA(A1:A10) ‚Üí non‚Äëblank cells.

üìç 4. Cell References

Relative: A1

Changes when copied. If C1 has =A1+B1, copying to C2 gives =A2+B2.

Absolute: $A$1

Locks reference. =B2 * $A$1 (A1 holds tax rate) ‚Äì copy down keeps $A$1.

Mixed: $A1 or A$1

$A1 ‚Äì column absolute, row relative. A$1 ‚Äì row absolute, column relative.

Multiplication table example

In B2: =$A2 * B$1 ‚Äì copy to entire table. Mixed references at work.

F4 toggles reference types while editing a formula.

Other sheets/workbooks

=Sheet2!A1 or ='[Budget.xlsx]Jan'!C10

Best practices

  • Use cell references, not hard‚Äëcoded numbers

  • Named ranges: =SUM(SalesData) * TaxRate
  • Add comments with N("comment")
