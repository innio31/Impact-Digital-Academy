2.1 What is an Excel Workbook?

An Excel workbook is the actual file you create and save on your computer. Think of it as a digital notebook or a binder. Just like a binder can hold many pages, an Excel workbook can contain many individual pages. The file extension for an Excel workbook is .xlsx. When you start Excel and select "Blank Workbook," you are creating a new, empty file to work in.

  • Primary container for all your data and analysis
  • Can hold one or more worksheets, chart sheets, and macro sheets
  • Save via File > Save As (choose location, name, click Save)
