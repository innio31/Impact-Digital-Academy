Real-World Collaboration Scenario

Scenario: Quarterly Report Review

You are part of a team preparing the Q3 Financial Report. The document has been passed between team members for review. Your role is to:

  1. Add comments with suggestions for improvement
  2. Track your own edits so others can see what you changed
  3. Review comments and changes from other team members (simulated in this assignment)
  4. Make decisions about which changes to accept or reject
  5. Finalize the document for distribution
  6. Lock the document to prevent further changes before sending to management

This scenario mirrors common workplace collaboration workflows. The skills you practice here are directly applicable to real-world document management.

MO-100 Exam Tips: Collaboration

  • Know the shortcuts: Memorize Ctrl+Alt+M (comment), Ctrl+Shift+E (track changes)
  • Understand markup views: Questions often ask about the visual differences between Simple, All, No Markup, and Original
  • Reviewer filtering: Practice viewing changes from specific reviewers
  • Accept/Reject options: Know how to accept/reject individual changes, all changes, or changes from specific reviewers
  • Lock Tracking: Understand when and why to lock change tracking

Frequently Asked Questions
What if I accidentally accept or reject the wrong change?
You can use Undo (Ctrl+Z) immediately after accepting/rejecting. However, if you've closed the document, the action is permanent. Always save a copy before bulk accepting/rejecting changes.
How do I see changes from only one reviewer?
Go to Review tab ‚Üí Show Markup ‚Üí Reviewers ‚Üí deselect all reviewers except the one you want to see. This filters both comments and tracked changes.
Can I recover a resolved comment?
No, resolving a comment is permanent. The comment remains but is grayed out and cannot be unresolved. If you might need it again, consider leaving it active or copying the text elsewhere before resolving.
What happens if I forget the Lock Tracking password?
There is no password recovery option in Word. If you forget the password, you cannot unlock tracking. You would need to work on a copy of the document without the lock, but this would break the edit history. Always store passwords securely.
How can I tell if Track Changes is on without looking at the Review tab?
Check the status bar at the bottom of the Word window. It displays "Track Changes: On" when enabled. You can also customize the status bar to always show this indicator.
