Assignment Overview

In this assignment, you'll apply document inspection techniques to clean and prepare a professional report. You'll use the Document Inspector, Accessibility Checker, and Compatibility Checker to identify and fix issues, then document your process. This assignment simulates real-world document preparation and reinforces skills tested on the MO-100 exam.

Total Points: 100 | Estimated Time: 2-3 hours

Learning Objectives
  • Use Document Inspector to find and remove hidden properties and personal information
  • Run Accessibility Checker and fix all accessibility issues
  • Check document compatibility and resolve compatibility warnings
  • Create accessible documents with proper structure and alt text
  • Document your inspection process and justify your fixes
  • Prepare for the MO-100 exam through hands-on practice

Frequently Asked Questions
What if I accidentally remove something important?
Always save a copy of your original document before using the Document Inspector. The "Remove All" option is permanent, so work on a copy.
How do I know if my alt text is good enough?
Good alt text conveys the same information as the image. For a chart, include the key insight. For a photo, describe what's important. If someone couldn't see the image, would they miss important information?
What if I can't fix all compatibility issues?
Document which issues you couldn't fix and explain why. Sometimes newer features don't have equivalents in older versions. In these cases, consider if you can achieve the same effect with simpler formatting.
Can I use a different sample document?
It's recommended to use the provided sample document as it contains specific issues we want you to find. If you must use your own document, ensure it contains comments, tracked changes, images, tables, and multiple pages.
How long should my reflection be?
Aim for 500-750 words (about 1-2 pages). Be thorough in explaining your process and what you learned.
