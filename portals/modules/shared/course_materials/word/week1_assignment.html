Objective

Instructions

Follow the steps below to create a one-page professional document. Complete all tasks in Microsoft Word 2019 (or later).

Part 1: Create & Save a New Document

  1. Open Microsoft Word and create a new blank document.
  2. Type the following title at the top:
    "Personal Learning Goals for This Course"
  3. Below the title, add today's date.
  4. Save the document to your computer with the filename:
    LastName_FirstInitial_W1_Assignment
    Example: Smith_J_W1_Assignment
  5. Save it first in the default .docx format.

Part 2: Use Navigation & Editing Tools

  1. Type at least three learning goals you have for this Word course.
    Example: "I want to learn how to create professional reports."
  2. Use the Navigation Pane (under the View tab) to find the word "learn" in your document.
  3. Use Go To (Ctrl+G) to jump to Page 1 (though you only have one page).
  4. Turn ON formatting symbols (Show/Hide ¬∂) to see spaces and paragraph marks.
    Hint: The button is in the Home tab, Paragraph group.
    Take a screenshot of this view.
  5. Turn formatting symbols OFF before moving on.

Part 3: Save in Another Format & Set Print Options

  1. Save a copy of your document as a PDF using Save As.
    Name it: LastName_FirstInitial_W1_Assignment.pdf
  2. Open the Print Preview (File > Print).
    Set the following:
    • Orientation: Portrait
    • Margins: Normal
    • 1 page per sheet
  3. Do not actually print, just review the preview.

Part 4: Prepare for Sharing

  1. At the bottom of your Word document, add this note:
    "Document prepared by [Your Name] for submission in Week 1."
  2. Use Share > Email as Attachment simulation:
    • Go to File > Share > Email
    • Choose "Send as Attachment" (but do not actually send an email)
  3. Close Word properly.
