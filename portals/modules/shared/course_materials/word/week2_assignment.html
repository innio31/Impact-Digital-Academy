Practice Exercise

Create a Newsletter with Sections and Columns

Activity: Follow these step-by-step instructions to build a professional-looking newsletter that demonstrates your mastery of section breaks, column layouts, and advanced formatting.

1 Open a new document and type a title: "Community News"

Start Microsoft Word 2019 and create a new blank document. Type the title at the top of the page.

Tip: Make sure you're using a standard font like Calibri or Times New Roman for professional appearance.
2 Format the title with advanced styling
  1. Apply Title style: Select the title text, then go to Home tab → Styles group → Title
  2. Add a Shadow text effect: With title selected, go to Home tab → Font group → Text Effects and Typography → Shadow → Outer: Offset Bottom
  3. Center align: Click the Center alignment button in the Home tab

Your title should look similar to this:

Community News
3 Type an introduction paragraph

Press Enter twice after the title, then type a brief introduction to your newsletter (3-4 sentences).

Welcome to the Community News newsletter! In this edition, we highlight recent events, community initiatives, and upcoming opportunities. Our goal is to keep residents informed and connected. Thank you for being part of our vibrant community.
4 Apply a two-column layout

Place your cursor in the introduction paragraph, then:

  1. Go to Layout tab → Page Setup group → Columns
  2. Select "Two" from the dropdown menu
Alternative method: You can also select "More Columns" for additional options like column width and spacing adjustments.

Column 2
5 Add a section break

After the introduction paragraph:

  1. Place cursor at the end of the introduction
  2. Go to Layout tab → Page Setup group → Breaks
  3. Select "Continuous" under Section Breaks
← Section Break (Continuous) →

The section break allows you to have different column formatting in different parts of your document.

6 Change column formatting in the new section

With your cursor in the new section (after the section break):

  1. Go to Layout tab → Page Setup group → Columns → More Columns
  2. In the Columns dialog box, select "Three"
  3. Check the box for "Line between"
  4. Click OK

7 Add sample text in each column

Type at least two paragraphs of text in each of the three columns. You can use placeholder text or write about community topics.

Content suggestion: Write about different topics in each column: Column 1 - Local Events, Column 2 - Community Projects, Column 3 - Volunteer Opportunities.
8 Insert a column break

To control where text flows between columns:

  1. Place cursor in the middle of the text in the second column
  2. Go to Layout tab → Page Setup group → Breaks
  3. Select "Column" under Page Breaks

This will force the text after the cursor to move to the next column.

Keyboard shortcut: Ctrl+Shift+Enter also inserts a column break.
9 Use Format Painter

Copy formatting from one paragraph to another:

  1. Select the introduction paragraph with your desired formatting
  2. Click the Format Painter button in the Home tab (Clipboard group)
  3. Click on another paragraph to apply the same formatting
Double-click trick: Double-click the Format Painter button to apply formatting to multiple paragraphs. Click it again or press Esc to turn it off.
10 Save your document

Save your completed newsletter with the filename:

YourName_Week2_Newsletter.docx

Example: JaneSmith_Week2_Newsletter.docx

Best practice: Save your work frequently as you progress through the steps to avoid losing any changes.

Section breaks allow you to change column formatting within the same document. For example, you might have a single-column title and introduction, followed by a two-column main content area, and a three-column footer with contact information.

Column breaks force text to move to the next column, which is useful for controlling where content appears. This is particularly helpful when you want to ensure a heading appears at the top of a new column.
